01 Color Tokens

Sistem warna berpusat pada keseimbangan antara Emerald Green sebagai aksi primer yang berenergi dan Deep Teal sebagai otoritas tenang. Setiap token harus dirujuk via CSS variable, bukan nilai hex langsung.

Emerald Green --primary: #10b981 Tombol utama, focus ring, ikon aktif, badge hijau
Forest Green --primary-dark: #0c8a60 Hover/active state, gradasi tombol, aksen teks
Mint Tint --primary-light: #d1fae5 Background badge, hover state ringan, focus area
Deep Teal --text-primary: #0b222c Semua judul (h1–h3), teks nama spot, heading nav
Mid Teal #1a3d4d Gradasi tombol primer, header merchant dashboard
Slate --text-secondary: #64748b Teks paragraf, deskripsi spot, label form, metadata
Soft Blue-Gray --bg-base: #f0f4f7 Background dasar halaman + radial gradient
Pure White --surface: #ffffff Card surface, form input, panel konten
💡 Warna semantik pendukung: #ef4444 (Error/Hapus), #f59e0b (Warning/Pending), #3b82f6 (Info/Link) — hanya digunakan untuk konteks status, bukan dekorasi.

11 Motion & Animasi Tokens

Animasi mengkomunikasikan hierarki dan respons. Semua token durasi dan easing distandarisasi agar seluruh UI terasa kohesif dan berkarakter sama.

--ease-spring
cubic-bezier(0.23, 1, 0.32, 1)
Tombol, kartu, FAB — semua aksi langsung pengguna. Terasa "berat dan memantul".
--ease-smooth
cubic-bezier(0.4, 0, 0.2, 1)
Transisi halaman, sidebar, modal — perpindahan tenang dan profesional.
--ease-bounce
type: "spring", stiffness: 400, damping: 30
Toast notification, dialog konfirmasi — playful namun tetap responsif.
--dur-fast
0.2s (200ms)
Feedback langsung: hover badge, perubahan warna border, opacity toggle.
--dur-normal
0.4s (400ms)
Animasi tombol, slide dropdown, focus state input form.
--dur-slow
0.6s (600ms)
Transisi halaman penuh, skeleton loading, entrance kartu pertama.
✦ Selalu hormati @media (prefers-reduced-motion: reduce) — nonaktifkan semua animasi dekoratif untuk aksesibilitas pengguna dengan sensitivitas gerak.
